============Display Elements===========

Display Elements-দুই প্রকার। যথা-

  • Inline.
  • Block.

Display Elements Example :-

[<যদি কোনো ট্যাগ ব্যবহার করার পর সে পুরো লাইন দখল করে রাখে তখন তাকে Block Elements বলে।] যেমন-

Display Elements Example :-

[<যদি কোনো ট্যাগ ব্যবহার করার পর সে পুরো লাইন দখল না করে নির্দিষ্ট জায়গা দখল করে রাখে তখন তাকে Inline Elements বলে।] যেমন-
